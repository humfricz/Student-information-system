: By entering particular Batch (Ex: 2015) and Grade ( EX:10.0,A+:9.5,A:9.0,B+:8.5,B:8.0,B-:7.5,C:7.0 ) click on Enter button will store the information in database.

: By clicking on upload students button you will be redirected to file upload page where user can upload student enrollment list click on Enter button will store the information of enrollment in database.

Note*: You need to upload only csv file in a given format. If not your file cant be uploaded.

: By clicking on upload courses button you will be redirected to file upload page where user can upload courses enrolled list and a click on Enter button will store the information of enrolled courses in database.

Note*: You need to upload only csv file in a given format. If not your file cant be uploaded.

: By clicking on upload scores button you will be redirected to file upload page where user can upload scores of students in enrolled courses list and a click on Enter button will store the information of scores in database.

Note*: You need to upload only csv file in a given format. If not your file cant be uploaded.

: By clicking on change password button you will be able to change password if you want change password and changed password will be updated in database.
Note*: use passwords which can be easily remembered.

: By clicking on Logout button you will be redirected to landing page.
